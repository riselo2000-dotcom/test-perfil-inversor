# Test de Perfil de Inversor

Herramienta interactiva basada en el cuestionario oficial de Vanguard. Calcula tu perfil de inversor a través de 11 preguntas y te recomienda una cartera de ETFs UCITS para implementar tu asset allocation.

## Características

- 11 preguntas oficiales de Vanguard, agrupadas por bloques temáticos
- Contexto educativo en cada pregunta
- Cálculo automático de puntuación y asignación a uno de los 9 perfiles oficiales
- Cartera concreta de 4-6 ETFs UCITS por perfil (Irlanda/Luxemburgo, acumulación)
- Datos verificados en justETF.com
- Exportación de resultados en Markdown
- Conexión directa con Curvo.eu para backtesting

## Stack

- React 18 + Vite
- Tailwind CSS
- Lucide Icons

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
